## 新手入门推荐路径

### 完全不想折腾 → 纯网页版
ChatGPT / Claude / 豆包 → Midjourney / 可灵 → Suno

### 想本地跑图像 → 推荐 Stability Matrix
一键安装 ComfyUI，不用自己配环境，模型自动下

### 想本地跑大模型聊天 → 推荐 Ollama
一行命令安装，再一行命令拉模型，5 分钟搞定

### 程序员 → 必装
Cursor / Claude Code + Git + Node.js + Python
