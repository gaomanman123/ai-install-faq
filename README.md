## 📦 一键安装

### 方法一：命令行安装（推荐）

打开终端，一行命令搞定：

```bash
git clone https://github.com/你的用户名/ai-install-faq.git ~/.agents/skills/ai-install-faq
```

然后**重启豆包**就可以用了！

### 方法二：手动安装

1. 下载本仓库 ZIP 包
2. 解压得到 `ai-install-faq` 文件夹
3. 放到豆包 Skill 目录：
   - **Mac**：`~/.agents/skills/`
   - **Windows**：`%USERPROFILE%\.agents\skills\`
4. 重启豆包
